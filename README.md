# FlowBoard – Task Manager Web App

FlowBoard is a simple and elegant productivity web application that helps users organize tasks using a Kanban-style workflow. It allows users to create tasks, move them through different stages of progress, archive completed tasks, and stay focused using a built-in Pomodoro timer.

This project was built as part of the **100 Days of Python Bootcamp** to practice Flask web development, authentication, database management, and UI design.

## Features

- User registration and login
- Personal task board for each user
- Kanban workflow:
  - To Do
  - Doing
  - Done
- Task status updates
- Task editing
- Task archiving with history tracking
- Pomodoro focus timer
- Flash notifications
- Responsive design
- Animated UI with particles and gradient effects

## Installation

Clone the repository:

git clone https://github.com/yourusername/flowboard.git cd flowboard

Create a virtual environment:

python -m venv venv source venv/bin/activate

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Then open your browser and go to:

http://127.0.0.1:5000
